# Automation & Web Scraper Bot

A modular Automation & Web Scraper Bot built with Python. Supports scheduled scraping, browser automation (Selenium optional), CLI usage, and structured outputs (JSON/CSV). Example scrapers, scheduler, Dockerfile, and tests included.

## Quickstart

### Setup (local)
```bash
git clone <this-repo-url>
cd auto-web-scraper-bot
python -m venv .venv
source .venv/bin/activate  # on Windows: .venv\Scripts\activate
pip install -r requirements.txt
